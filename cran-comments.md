# cran-comments.md

This is a development conversion, not a CRAN submission. Newly authored package
code is GPL (>= 2), but the pinned historical XCNV repository has no license
grant. Its model, training data, scripts, and annotations are excluded from the
package tarball and remain redistribution
blockers. The independently authored ACMG/ClinGen 2019 scoring lane uses only
synthetic test data in the package; scientific resource packs remain external
and provenance-labelled.

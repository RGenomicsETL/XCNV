# XCNV: Genome-Wide Copy Number Variant Pathogenicity Prediction

An R-native interface to the X-CNV algorithm published by Zhang et al.
(2021). The published tree is bundled in a runtime-neutral form; large
annotation resources are supplied by the user. The package also provides
an independently authored, provenance-first implementation of the
ACMG/ClinGen 2019 constitutional CNV scoring contract.
